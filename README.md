# dot-emacs-dot-d
Personal .emacs.d configuration

-------- Elisp ------------

read-only-mode toggle (C-x C-q)

C-u 0 M-x byte-recompile-directory

-------- General Programming -----

comment M-;
paren matching C-M-n and p like % in vi

vc-diff or vc-ediff for git diff

--------- GO --------

	M-> jump in
	M-< jump out




--------- HTML ------------

C-c r|l-arrow tag matching like % in vi
